System Requirements

To run Godot Engine 4.x, you need:

  • OS: Windows 10, macOS 10.12+, or Linux (any recent distro)
  • CPU: Dual Core CPU, 64-bit (Intel/AMD or ARM)
  • RAM: 4 GB or more recommended
  • Graphics: OpenGL 3.3 compatible hardware (Vulkan support recommended for 3D)
  • Storage: ~500 MB for the engine itself
